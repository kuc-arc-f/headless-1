### Summary

headless CMS sample, Next.js + mongodb

***
### required
* Next.js : 10.0.0
* mongodb : 3.6.3
* node : 14.11


***
### Setup

npm install

***
### setup , etc
* next.config.js

BASE_URL: "http://localhost:3001"

* package.json / scripts

"dev": "next dev -p 3001"

***
### start server
* Start :

yarn dev

* build:

yarn deploy
