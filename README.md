### Summary

headless CMS sample, Next.js + mongodb

***
### required
* Next.js : 10.0.0
* mongodb : 3.6.3
* node : 14.11

***
### setup , etc
* next.config.js

BASE_URL: "http://localhost:3001"

* package.json / scripts

"dev": "next dev -p 3001"

***
### Setup

npx create-next-app app1 --use-npm --example "https://github.com/vercel/next-learn-starter/tree/master/learn-starter"

***
### start server
* Start :

yarn dev

* build:

yarn deploy
